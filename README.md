# Generator Capability Curve and P-Q Analysis
## Overview
This project focuses on the analysis of generator capability curves, P-Q operating regions, torque-angle characteristics, and V-curve behavior in power systems using MATLAB. The project investigates synchronous generator performance under different operating conditions, including field current variation, reactive power limits, stability analysis, and power factor behavior.

## Features
- Generator capability curve analysis
- P-Q operating region visualization
- Torque-angle and stability analysis
- V-curve analysis for multiple power levels
- Reactive power and field current evaluation
- MATLAB-based power systems simulation and plotting

## Tools Used
- MATLAB

- MATLAB source files (.m)
- Capability curve analysis scripts
- V-curve analysis programs
- Torque-angle analysis code
- Simulation plots and outputs
